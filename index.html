About TOKAD

TOKAD is a powerful tool for tokenizing Algerian dialect text. It helps you break down text into meaningful tokens for analysis and processing.

Help

Need assistance? Here's how to use TOKAD:

  • Enter text in the input box and click "Segment Text".
  • Upload a .txt file and click "Process File".
  • View the segmented results in the output area.
